SkySkin AeroDetect Patch – Aircraft Vibration Monitoring System

Technologies Used: Arduino Nano, MPU6050, BMP280, Python

SkySkin AeroDetect Patch is a real-time aircraft vibration monitoring system developed using MEMS sensors and Arduino Nano. The system collects vibration data and processes it using Python to classify structural conditions as Normal, Warning, or Abnormal. This project focuses on improving aircraft structural safety through continuous monitoring and data analysis.
